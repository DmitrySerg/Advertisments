# Advertisments
Предсказание частоты просмотра объявлений по его описанию

С avito.ru была выкачана база объявлений по собакам по состоянияю на 13.06.2016. Поля для скачивания:

- *description* - описание объявления
- *id* - идентификационный номер объявления
- *num_views* - количество просмотров 
- *price* - цена
- *create_date* - дата создания объявления
